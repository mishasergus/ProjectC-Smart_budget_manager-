Smart Budget Manager 💰
Smart Budget Manager — це десктопний додаток для керування особистими фінансами, розроблений на мові C++ з використанням технології Windows Forms (C++/CLI). Проект дозволяє користувачам відстежувати доходи та витрати, категоризувати транзакції та забезпечує надійне зберігання даних за допомогою SQLite та шифрування.

🚀 Основні можливості
Авторизація та безпека: Система реєстрації та входу користувачів із хешуванням паролів за допомогою алгоритму SHA-256 (OpenSSL).

Управління транзакціями: Створення, редагування та видалення записів про витрати та доходи.

Категоризація: Групування фінансових операцій за категоріями для кращого аналізу.

База даних: Використання SQLite для швидкого та локального зберігання всієї інформації.

Логування: Система відстеження подій та помилок у файл log.txt.

Інтуїтивний інтерфейс: Графічне вікно (GUI) на базі Windows Forms для зручної взаємодії.

🛠 Технологічний стек
Мова програмування: C++ (Standard 20)

GUI Фреймворк: C++/CLI (Windows Forms)

База даних: SQLite 3

Криптографія: OpenSSL (SHA-256)

Інструменти: Visual Studio 2025

📂 Структура проекту
LogForm.h — Основна логіка інтерфейсу користувача та обробка подій форми.

API.h / APILogic.cpp — Ядро системи: взаємодія з базою даних SQLite (CRUD операції).

ShifrClass.h / ShifrRealisation.cpp — Клас для шифрування та безпеки даних.

UserClass / TransactionClass / CategoryClass — Моделі даних для об'єктно-орієнтованого представлення сутностей.

logger.h — Простий інструмент для запису системних логів.

Wrapper.h — Забезпечує зв'язок між керованим (managed) та некерованим (native) кодом.

⚙️ Встановлення та запуск
Попередні вимоги:
Visual Studio 2025 (з встановленим робочим навантаженням ".NET Desktop Development" та підтримкою C++/CLI).

Бібліотека OpenSSL.

SQLite (включено в проект).

Кроки для запуску:
Клонуйте репозиторій.

Відкрийте файл рішення .sln у Visual Studio.

Переконайтеся, що шляхи до sqlite3.h та бібліотек OpenSSL у налаштуваннях проекту вказані вірно (наразі використовуються абсолютні шляхи, які можуть потребувати коригування під вашу систему).

Скомпілюйте проект (Build Solution).
